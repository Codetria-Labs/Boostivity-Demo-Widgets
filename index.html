Tu sitio web de Prueba

Esta es una página estática para simular cómo se vería el widget instalado en una página real de tus clientes.

ℹ️ Cómo probar el widget aquí

1. Entrá al panel de Admin de Boostivity en Integrations > Widget.
2. Copiá el código HTML (Snippet) que aparece ahí.
3. Abrí este archivo (Boostivity-Widget/demo.html) en tu editor de código.
4. Pegá el código justo antes del cierre de la etiqueta </body> abajo de este archivo.
5. Entrá a http://localhost:9090/demo.html y debería aparecer el bot.
